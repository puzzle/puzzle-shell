## More Variants

If you need more variants, you can add them indiviually using the [fontsource.org](https://fontsource.org/) font packages.

For example, if you need a light variant, install the `@fontsource/roboto` package in your project and import the variant in your CSS:

```css
import '@fontsource/roboto/300.css';
import '@fontsource/roboto/300-italic.css';
```
