# Container

The `pzsh-container` component is the top-level component that implements the page layout. It contains all other Puzzle Shell components as well as the application's or site's content.

Example usage:

```
<pzsh-container>
  <pzsh-topbar>...</pzsh-topbar>
  <pzsh-banner>...</pzsh-banner>
  <pzsh-hero>...</pzsh-hero>
  <main>The content</main>
  <pzsh-footer>...</pzsh-footer>
<pzsh-container>
```
