# Icons

The Puzzle Shell uses the [Unicons](https://iconscout.com/unicons) icon set, which is Apache 2.0 licensed.

It provides the following basic set of icons via the `pzsh-icon` component:

<div>
  {Object.keys(icons).map(icon =>
    <div style={{display: "flex", alignItems: "center",}}>
      <pzsh-icon name={icon} style={{marginRight: "0.7rem"}} />
      <code>{`<pzsh-icon name=\"${icon}\"></pzsh-icon>`}</code>
    </div>)}
</div>

You can grab additional icons from the [full list of icons](https://iconscout.com/unicons/explore/line) and use them as SVGs in your project.

## Optimization

When downloading icon SVGs, make sure to process them as follows:

- Optimize them with [SVGOMG](https://jakearchibald.github.io/svgomg/):
  - Copy the SVGs markup
  - Goto [SVGOMG](https://jakearchibald.github.io/svgomg/) and use "Paste markup"
  - Copy the reduced SVG (copy icon on the bottom right)
